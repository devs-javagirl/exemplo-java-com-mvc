 
## Start na aplicação:
 - Abrir o arquivo "Boot.java" > botão direito no arquivo > "Run as" > "Java application" (ou "Spring Boot App")
 
### Para acessar a aplicação:
 - http://localhost:8080/
 
### Para acessar console do banco:
 - http://localhost:8080/h2-console/
 - URL: jdbc:h2:mem:testdb
 - User name: sa
 - Sem senha
